# Rock Paper Scissors Game (Python Tkinter GUI)

## Description

This project is a graphical Rock Paper Scissors game developed using Python and the Tkinter library. The player competes against the computer, which randomly selects Rock, Paper, or Scissors. The application keeps track of scores and provides options to play again or reset the game.

## Features

* User-friendly GUI built with Tkinter
* Three buttons for Rock, Paper, and Scissors
* Random computer selection
* Game result display (Win, Lose, or Tie)
* Score tracking for both player and computer
* Play Again functionality
* Reset Score option
* Simple and interactive interface

## Requirements

* Python 3.x
* Tkinter library (included with standard Python installation)

## Installation

1. Download or copy the Python source code.
2. Save the file as:

```
rock_paper_scissors.py
```

3. Open a terminal or command prompt.
4. Navigate to the folder containing the file.
5. Run the program:

```bash
python rock_paper_scissors.py
```

## How to Play

1. Launch the application.
2. Click one of the buttons:

   * Rock
   * Paper
   * Scissors
3. The computer will automatically make its choice.
4. The game determines the winner according to the rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
5. The result and updated scores are displayed.
6. Use the **Play Again** button to start another round.
7. Use the **Reset Score** button to clear all scores.
